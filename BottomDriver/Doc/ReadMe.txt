
Программа для управления нижним двигателем

main():
    инициализация
    разбор посылок по COM-порту
    управление GPIO для ШИМа

Прерывания

MotorControlIsr():
    Определение положения относительно арретира, метки для EEPROM
    Задание значений для ПИД-регулятора, вычисление скорости
    Задание момента двигателя при помощи библиотеки

SciRxIsr():
    Приём и обработка байта, принимаемому по COM-порту

CpuTimerIsr():
    Инкрементирует счетчики
    1 секунда = 16 тиков


Алгоритм перепрошивки:
	Прошивка принимается по SCI в буффер firmwareBuffer в отдельной области RAM
	По принятию целой прошивки, прыгает в Загрузчик (другой проект)
	В Загрузчике проверяется наличие новой прошивки в отдельной области RAM
	Если есть прошивка, то прошивает
	Если нет прошивки, то прыгает в основную прошивку
	

Build Configurations:

	RAM - режим работы в RAM
	
	FLASH - обычный режим работы во FLASH
	
	FLASH_WithLoader - прошивка с возможностью прошивки 
		Сначала эту прошиваем, затем прошиваем загрузчик
		
	FLASH_PrepareFirmware - Подготовить прошивку для передачи
